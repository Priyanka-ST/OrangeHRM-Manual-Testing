# OrangeHRM Manual Testing Project

## Project Overview
This project demonstrates manual testing activities performed on the OrangeHRM application.

## Application Under Test
- Application Name: OrangeHRM
- Testing Type: Manual Testing
- Testing Level: System Testing

## Testing Artifacts
The following testing documents are maintained as part of this project:
- Requirement Analysis
- Test Plan & Scope
- Test Scenarios
- Test Cases
- (Future) Defect Report
- (Future) Test Execution Report

## Tools Used
- MS Word
- MS Excel
- GitHub

## Objective
To design and document complete manual testing artifacts following the Software Testing Life Cycle (STLC).
